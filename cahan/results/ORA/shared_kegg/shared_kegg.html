  • Enrichment method: ORA
  • Organism: dmelanogaster
  • Enrichment Categories: pathway_KEGG

  • The interesting list contains 2494 user IDs in which 2273 user IDs are unambiguously mapped to 2273 unique entrezgene IDs and 221 user IDs can not be mapped to any entrezgene ID.
  • The GO Slim summary are based upon the 2273 unique entrezgene IDs.

Parameters for the enrichment analysis:

  • Minimum number of IDs in the category: 5
  • Maximum number of IDs in the category: 2000
  • FDR Method: BH
  • Significance Level: Top 10
